CIIADHEL RESTful API


Versión 1.0.0

En la versión preliminar V1.0.0 se presentan las rutas (aeropuertos - usuarios - notificaciones - Archivos - Favoritos) con sus respectivas acciones. A continuación, se presentan con mayor detalle el funcionamiento de estas acciones y la información que retornan. En el siguiente botón puede encontrar una información más detallada con aspectos de consumo en diferentes lenguajes de programación y aspectos mas técnicos.

Esta es la ruta de los Autenticacion (Users) donde contiene las siguientes acciones:

  1. Iniciar sesion

    2. POST: https://ciiadhel-api-rest.herokuapp.com/api/users/login/cedula/password

    • Parámetros:
      1. Cedula (String)
      2. Contraseña (String)
